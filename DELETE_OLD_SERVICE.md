# 如何删除 Render 上的旧服务

## 方法 1：通过 Dashboard 删除

1. 登录 Render Dashboard：https://dashboard.render.com
2. 找到 "wecom-accounting-bot" 服务
3. 点击服务名称进入详情页
4. 点击右上角的 **"Settings"**（设置）
5. 滚动到底部，找到 **"Danger Zone"**（危险区域）
6. 点击 **"Delete Service"**（删除服务）
7. 确认删除（输入服务名称确认）

## 方法 2：通过服务列表删除

1. 在服务列表中，找到 "wecom-accounting-bot"
2. 点击服务右侧的 **"..."**（三个点菜单）
3. 选择 **"Delete"**（删除）
4. 确认删除

## 删除后

删除旧服务后，可以：
1. 创建新的服务用于微信公众号
2. 或者直接修改现有服务的配置
